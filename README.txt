Paper title: StretchDenoise: Parametric Curve Reconstruction with Guarantees
by Separating Connectivity from Residual Uncertainty of Samples
Paper authors: Stefan Ohrhallinger and Michael Wimmer
Operation system required: Ubuntu 16.04LTS
License: GNU General Public License (see gpl.txt)

Download the source code:
git clone https://github.com/stefango74/stretchdenoise/stretchdenoise.git
cd stretchdenoise

See BUILD.txt on how to configure and build the software.
Then run './figN.sh' or './tabN.sh' to generate the corresponding figure or table.

Note that a screen size of at least 768x768 is required, otherwise the figures may become squashed.
